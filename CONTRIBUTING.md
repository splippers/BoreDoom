# Contributing

This project is intentionally public and under a permissive license so that:
- Anyone can fork it
- Improve the AR tracking
- Add new chore modes
- Ship something people can actually use

## Ways to help
- Implement real coverage mapping (indoor + outdoor)
- Improve scoring formulas
- Add new chore modes (mopping, leaf blowing, etc.)
- Polish AR HUD and UX
- Integrate health/activity metrics

## PR guidance
- Keep PRs focused and documented
- Prefer additive, modular mode implementations over editing core systems
